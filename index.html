Intelligent Motorized Racecar

This was a semester-long project that involved programming an intelligent motorized racecar that could autonomously drive itself around a race track. The main controller for the car was a FRDM K64F microcontroller that connected to a motorshield to drive two DC motors, a linescan camera, and a steering servo. A full IEEE-standard paper was written and is available in the project source.

This car won first place in the competition against other classmates (humble brag).

Median Filter

This project showcases a median filter implemented on a Nexys4 DDR FPGA. Matlab was used to send an image to the FPGA via UART, and the image was stored in DDR2 memory using AXI Stream/Lite interfaces. A pipelined median filter was designed to process the image.

A continuation of this project used MicroBlaze to compare a purely soft-core processor implementation against a hardware coprocessor's. The hardware coprocessor implementation had a speedup of 14 over the MicroBlaze implementation.

Soft MIPS Processor

This project was completed as part of a digital design course's semester-long labratory exercise. A soft core pipelined MIPS microprocessor was designed and implemented on a Basys-3 FPGA. Each stage in the pipeline was verified both in simulation and hardware.
